count the times 
1. use hashmap to count occurence when it is > n/2 then return O(N)
2. sort and return the mid value O(nlogn)

```cpp
    int majorityElement(vector<int>& nums) {
        unordered_map<int,int> mp;
        for(int n:nums) 
        {
            mp[n]++;
            if(mp[n]>nums.size()/2) return n;
        }
        return 0;
    }
```
